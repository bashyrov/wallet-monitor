Changelog

What we ship, when, why. Newest on top.

2026-05-17 v0.6

Trade engine: Kraken spot + Hyperliquid spot + Lighter

  • Kraken spot + Hyperliquid spot adapters — 13 venues now do spot trading.
  • Lighter trading enabled via official Go SDK (Schnorr + Poseidon over ECgFp5). 18/18 perp venues now trade through Go.
  • Faster funding for slow venues: OKX 60s → 30s, Paradex 5min → 5s, Lighter 5min → 10s.
  • OG link previews for /arb, /screener, /portfolio. Per-pair preview on /arb shows symbol + venues when shared in messengers.
  • /landing retired — homepage IS the landing now (301 from /landing → /).
2026-05-15 v0.5

Service Worker + scroll perf + WS idle-disconnect

  • Service Worker — repeat visits load in ~50ms instead of ~500ms. Offline-tolerant: HTML shell + last data render even if the network is down.
  • content-visibility: auto on orderbook + history rows — 5-10× scroll FPS on mobile.
  • WS idle-disconnect after 5 min of no activity. Tab in background = 0% client CPU. Reconnects on any user input.
  • Trade panel intervals tightened: balance check 15s → 5s, positions 8s → 3s, PnL 60s → 15s.
  • OKX 500ms post-order fill-fetch removed — orders return immediately.
2026-05-14 v0.4

API latency sprint: snapshot caches everywhere

  • /api/screener/funding p50: 702ms → 11ms (snapshot cache + background warmer).
  • /api/screener/exchange-health: 1440ms → 2.7ms.
  • /api/auth/me: 344-1043ms → 5-37ms (per-user 30s cache with invalidate on plan change).
  • Reconcile worker moved to Redis leader-election — 8 parallel workers → 1.
  • Funding feed health metrics now per-venue (ts_by_ex); status dots reflect real WS push freshness.
2026-05-13 v0.3

Volume coverage 96% → 99.7%

  • Ethereal: adapter ignored volume24h field — fixed, 18/18 instruments now have volume.
  • Binance: 126 SETTLING/PENDING_TRADING ghost rows filtered via exchangeInfo TRADING set.
  • Aster: same fix for 72 tokenized-equity rows.
  • Kraken: skip inactive PF_* perps with vol=0 AND last=null.
  • OKX funding fetch expanded from subscribed-only (~40) to all 319 swaps — screener sees 8× more OKX inventory.
2026-05-12 v0.2

Live trade-stream layer + 18-venue feed

  • New /ws/trades channel pushes individual trade fills from every venue (100-300 trades/sec on hot Binance pairs).

2026-05-09 v0.1

PnL backfill from venue fills

  • "Sync" button on the PnL tab pulls 7 days of fills + funding from each venue, materialises closed positions, auto-pairs spot/short basis pairs.
  • New tables: trade_fills (raw venue fills with idempotency), fills_sync_cursor (per-venue high-watermark).
