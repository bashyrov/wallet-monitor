FAQ

The straight answers

No marketing fluff — only what people actually ask before signing up.

01 How do you keep my API keys safe? +
Every credential you paste is immediately encrypted with Fernet (AES-128-CBC + HMAC-SHA256). The Fernet key is derived from your ENCRYPTION_KEY env variable via PBKDF2-HMAC-SHA256 with 260 000 iterations. The plaintext key never touches disk — only the encrypted blob lives in your Postgres. API responses always mask keys to abcd****wxyz.
02 Is Avalant really free? +
The Basic plan (4 wallets, full Screener access) is free forever — no credit card, no trial. Paid tiers exist for users who need more wallets or priority support, but the core product (Screener + Portfolio) works without spending a cent. Pricing on /pricing.
03 Do you route my trades? +
No. Ever. We only read data to show you balances, positions and arb opportunities — we never send orders on your behalf. Read-only API keys are recommended everywhere; trade adapters exist for users who want to query positions but are never given withdraw permission.
04 Which exchanges and chains do you support? +
11 CEX: Binance, OKX, Bybit, Gate.io, KuCoin, MEXC, Bitget, BingX, WhiteBIT, Kraken, Backpack. 4 perp DEX: Hyperliquid, Aster, Lighter, Ethereal (Paradex read-only). 13 chains: Ethereum, BSC, Polygon, Arbitrum, Optimism, Base, Avalanche, Tron, Solana, zkSync, Linea, Scroll, Mantle, Blast. Screener adds a live WebSocket orderbook stream for all 11 CEX + Hyperliquid + Aster.
05 How fresh is the Screener data? +
Orderbooks stream from each exchange via WebSocket — most pairs refresh in <500ms. Funding rates refresh every 6 seconds. The full arb list is recomputed and pushed to the UI every 3 seconds, with an explicit Live / 3s / 10s / Paused cadence control per user.
06 Can I self-host the whole thing? +
Yes — that's the default. git clone, cp .env.sample .env, docker compose up -d. Postgres, FastAPI backend, Nginx with Let's Encrypt. Detailed deployment docs in the repo. Zero vendor lock-in.
07 What about Paradex and Hyperliquid private keys? +
Paradex: we take a JWT / API token, not your Ethereum private key. Read-only by default.
Hyperliquid: an agent wallet private key — a separate keypair you generate on HL for API use, which can only trade your positions, not withdraw. Never your main wallet.
08 How do Telegram alerts work? +
Link your Telegram once via a one-time deep-link. Then from Profile or the Screener, set spread-threshold alerts per pair (e.g. HYPE HL→OKX at 0.15%/8h, any direction). Our backend polls every 60 s and fires via the Avalant bot — with per-alert cooldown so you don't get spammed.
09 Can multiple people use one account? +
Each account is single-user. You can run multiple accounts on one self-hosted instance — the first registered user becomes admin automatically. Team roles, shared wallets and per-team permissions are on the roadmap.
10 Where can I report bugs or request features? +
GitHub issues, the Telegram community, or directly through the app's support button. We read everything. Feature requests that match the product vision usually ship within a release cycle.
