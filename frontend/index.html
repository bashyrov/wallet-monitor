Features

Built for serious
crypto users

No compromises. Everything you need to track a multi-source portfolio — without giving up control.

🔗

Multi-source aggregation

Pull balances from CEX accounts via API, on-chain wallets by address, and perp DEX positions — all in one request, all in parallel.

⚡

Instant parallel fetch

All providers are queried simultaneously using asyncio. 8 exchanges + 5 chains responds in under 3 seconds, not 30.

🏷

Custom tags & filters

Create your own tags — by strategy, risk, person, or anything else. Filter wallets instantly. Group checks by tag.

🔒

Fully self-hosted

Your API keys and addresses never leave your server. No cloud, no third party, no tracking. SQLite, one command to start.

How it works

Three steps to full visibility

01
Add your wallets
Paste API keys for exchanges, or a wallet address for chains and perp DEXes. Supports read-only API keys — write access never needed.
02
Tag & organize
Create custom tags with colors. Assign multiple tags per wallet. Filter the dashboard to see only what matters right now.
03
Check balance
Select wallets with checkboxes — or check all at once. See individual breakdowns and a unified portfolio total with stablecoins separated.
